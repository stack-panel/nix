# My Project

## Getting Started

```bash
# Enter development shell
devenv shell

# Or with direnv (recommended)
direnv allow
```

## Stackpanel Features

This project uses [stackpanel](https://github.com/darkmatter/stackpanel) for:

- AWS Roles Anywhere authentication
- Step CA certificate management
- Secrets management

See `devenv.nix` to enable features.
